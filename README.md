1. Наличие opencv в корне диска C "C:\opencv"
2. Прописанный путь в переменных среды для "C:\opencv\build\x64\vc16\bin"

Подробнее: https://www.geeksforgeeks.org/opencv-c-windows-setup-using-visual-studio-2019/
